Iteration

Prompt & Umsetzung in AntiGravity

Das Interface wurde mit einem detaillierten System Prompt entwickelt — visuell angelehnt an die Kaisin-Website, aber als minimalistisches Gamification-Tool für Führungskräfte. Der Prototyp wurde bewusst agil entwickelt: zuerst als funktionales Logik-Modell, danach der markante Styleguide.

System Design Prompt (Auszug)

System Design & Branding Guidelines: Passe das User Interface der Web-Applikation „Commitment Poké-Bowl Lab" exakt an die digitale Design-Sprache von kaisin.ch an. Hintergrund-Grundton: Ein sehr weiches, warmes Off-White (#FDFBF7). Primäre Akzentfarbe: Ein sattes, leuchtendes Tomatenrot (#E32619). Typografie: Eine feine, geometrische, serifenlose Sans-Serif-Schrift — analog zum kaisin-Logo-Stil.

Reflexion

Learnings

Elianes Version des Commitment Poké-Bowl Labs hat funktioniert und zeigt einen überzeugenden iterativen Prozess — die spielerische Metapher macht Ressourcenallokation für Führungskräfte wirklich greifbar.

Mein Versuch mit dem Bewertungssystem hat leider nicht wie geplant funktioniert. Die Arbeit mit AntiGravity war deutlich anspruchsvoller als erwartet — besonders die Gestensteuerung war eine echte Herausforderung.

AntiGravity ist ein mächtiges aber anspruchsvolles Tool — ohne tiefes Verständnis der Gestenlogik kommt man schnell an Grenzen. Die Erkennung im Live-Betrieb musste mehrfach angepasst werden.

Learning: Bei komplexen Interaktionsmodellen zuerst mit einem einfacheren Ansatz starten, bevor Gestensteuerung eingebaut wird.
